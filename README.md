### ğŸŸ© Cara Registrasi WONDR by BNI 
**Untuk Nasabah Baru**
1. Buka aplikasi dan klik "Mulai Sekarang" atau "Yuk Aktifin di Sini".
2. Masukkan nomor HP yang terdaftar di BNI Mobile Banking.
3. Pilih metode pengiriman OTP (SMS/WhatsApp) dan masukkan kode yang diterima.
4. Siapkan E-KTP, lakukan pemindaian, dan pastikan data (NIK, nama, tanggal lahir) sesuai.
5. Ikuti petunjuk di layar, pastikan pencahayaan cukup dan tidak memakai masker, topi, atau kacamata tebal.
6. Buat password untuk login wondr by BNI (bisa sama dengan BNI Mobile).
7. Buat PIN wondr untuk kemudahan transaksi.
8. Daftarkan alamat email aktif untuk keamanan akun.
**Untuk Nasabah Lama**
1. Pilih kartu debit/rekening BNI yang ingin disambungkan.
2. Aplikasi akan melakukan sinkronisasi data dari BNI Mobile Banking.
3. Setelah semua langkah selesai, Anda sudah bisa bertransaksi menggunakan wondr by BNI.
